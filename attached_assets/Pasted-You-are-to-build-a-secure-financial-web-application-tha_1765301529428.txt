You are to build a secure financial web application that manages recurring monthly payments, upsells, targets, reporting and user accounts.
The application must follow the system architecture, sitemap, modules and functionalities below.

📍 PROJECT OVERVIEW

A private-access financial management system for tracking recurring monthly payments grouped by Region and Project Managers. The system should support dashboard analytics, monthly planning, reporting, and admin-level user management. Only invited users should be able to log in.

Stack (Preferred Example):

Frontend: React / Next.js / TailwindCSS

Backend: Node.js (Express) OR Python FastAPI

DB: PostgreSQL / MongoDB

Auth: JWT + bcrypt + secure email OTP reset

Deployable in Replit environment
